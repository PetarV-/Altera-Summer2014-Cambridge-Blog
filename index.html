Week 1

During the first week of our internship, we have:

  • Attended Altera's training session in High Wycombe;
  • Learnt the basics of Bluespec SystemVerilog and used it to program several example modules;
  • Got all of Quartus' tools to work by locating missing libraries, fixing rules for USB device permissions, etc;
  • Booted the ARM processor on the SoCKit using an SD card containing Linux;
  • Interfaced to the ARM in two ways: initially using Minicom over a USB connection, now also via Ethernet+SSH to ease file transfer.
  • Successfully programmed the FPGA using the ARM (on-boot) as well as JTAG.

  • Established basic communication ARM -> FPGA, in the following manner:

  • Put linux on an SD card and used that to boot the on chip arm processor (which we can communicate with over minicom)

  • Wrote a simple bluespec program take creates an avalon slave that takes integers and returns them multiplied by 3
  • Connected the triple program to a HPC component in Qsys
  • Compiled the binary for the FPGA and transfered it to the SD card
  • Wrote a C program that takes numbers on on the command line and writes them to the memory mapped hpc2fpga bridge
  • Cross compiled the C program for arm, transfered it and the FPGA bitstream to the SD card
  • Programmed the FPGA from the arm chip.
  • Ran the C program on the arm, supplying numbers on the command line across minicom, successfully getting the trippled value back

We hope to...

  • Achieve communication on the fpga2hpc bridge (hopefully directly accessing SDRAM, maybe peripherals?)
  • Get ethernet connectivity.
  • Decide on a project
